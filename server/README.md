## Чтобы запустить сервер выполните следующие команды:

```bash
mkdir build
```

```bash
cd build
```

```bash
cmake ..
```
### Билд проекта
```bash
make
```
### Запуск сервера
```bash
./server <port_number> <root_directory>
```
### Пример
```bash
./server 8080 /path/to/root/directory
```
